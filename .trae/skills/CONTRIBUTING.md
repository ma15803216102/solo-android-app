# 贡献指南

感谢你对 Koi-Prd-Skill-Suite的兴趣！

## 如何贡献

### 提交 Issue

如果你发现了 bug 或者有功能建议，请提交 Issue。

### 提交 Pull Request

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开 Pull Request


### 文档规范

- 每个技能必须包含 README.md 和 SKILL.md 文件
- README.md 包含技能的基本信息和使用方式
- SKILL.md 包含技能的详细定义和工作流程


## 行为准则

请遵守 [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) 中的行为准则。

## 许可证

你的贡献将被采用 MIT 许可证。
